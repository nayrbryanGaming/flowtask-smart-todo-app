# Key Performance Indicators (KPIs) & Growth Strategy

## 1. Initial User Acquisition Strategy
- **Beta Waitlist:** Utilize the Next.js landing page to collect emails before full launch. Offer "Lifetime Premium Beta" to the first 1,000 signups.
- **App Store Optimization (ASO):** Focus keywords: "ADHD task tracker", "Pomodoro to do list", "Productivity analytics".
- **Productivity Communities:** Grassroots launch in Subreddits (`r/productivity`, `r/ADHD`, `r/Entrepreneur`) and Discord servers focusing on studying/freelancing.

## 2. Content Marketing Plan "The Flow State Blog"
- **Medium/Dev.to Publications:** Write transparent "Build in Public" articles about using Flutter/Firebase to build the app.
- **TikTok/Reels:** Short-form content focusing on "Aesthetic Study Routines" and "How I manage my chaos as a freelancer." showcasing the app's dark mode and analytics.

## 3. Product Hunt Launch Strategy
- **Pre-launch:** Engage with makers, tease screenshots of the analytics dashboard (the core differentiator) on Twitter/X.
- **Launch Day:** Highlight the "Personal Productivity Intelligence" angle. Offer an exclusive PH promo code (`PHFLOW26`) unlocking 3 months of Premium.
- **Goal:** Top 5 Product of the Day.

## 4. Target First 10k Users Milestones
- **Month 1:** 1,000 users via PH and direct network.
- **Month 2-3:** Subreddit scaling, ASO kicks in (Target: 4,000 users).
- **Month 4-6:** Viral TikTok/Reels loops and referral system (Target: 10,000 users).

## 5. Core KPI Metrics to Track

### Acquisition
- **CPA (Cost Per Acquisition):** Target < $1.50 using targeted meta ads later.
- **Organic Install Rate:** via Play Store Search.

### Engagement (The most critical metric for to-do apps)
- **DAU (Daily Active Users):** Users who open the app AND complete at least one task or run one focus timer.
- **MAU (Monthly Active Users):** Baseline ecosystem size.
- **DAU/MAU Ratio (Stickiness):** Target > 25% (Productivity apps need high stickiness).

### Retention
- **D1 Retention:** Target 40% (Did the onboarding work?)
- **D7 Retention:** Target 20% (Did they form a habit?)
- **D30 Retention:** Target 10% (Long-term value unlocked).

### Monetization
- **Free-to-Paid Conversion Rate:** Target 3-5%.
- **LTV (Life Time Value):** Projected > $30 per premium user.
